Design system · v1.0

A calm, composable system for SoloHub.

Modern, minimal, and built for multi-tenant salons. One set of tokens, four surfaces, every tenant's brand on top.

Next.js 16 · Tailwind v4 Geist Sans WCAG AA
Foundations

Principles

Six rules that shape every decision in the system. When tokens and components conflict, principles win.

01

Calm by default

Salons are sensory businesses. The product UI should feel quiet so the owner's brand can be loud.

02

Content over chrome

Minimal borders, generous whitespace, soft surfaces. No decorative noise.

03

One brand, many tenants

Each tenant overrides a single primary color. Everything else stays neutral so branding reads intentional.

04

Accessible at rest

WCAG AA contrast, full keyboard paths, 44px touch targets — designed first, not retrofitted.

05

Composable, not rigid

Tokens into primitives, primitives into patterns. No one-off components.

06

Minimum viable motion

Motion confirms, never decorates. 150–250ms, soft easing, always respects reduced-motion.

Implementation

Practical notes for adopting the system in the current Next.js 16 + Tailwind v4 codebase.

1 · Drop in the tokens

Replace app/globals.css with design-system/tokens.css (or @import it). All tokens become available as Tailwind utilities (e.g. bg-brand-600, text-h2, rounded-lg).

2 · Build the primitives

Create components/ds/ with one file per component. Each wraps a native element, accepts variant, size, asChild, and forwards refs. Export a barrel components/ds/index.ts.

3 · Inject tenant brand at the root

In the tenant layout, inject --color-brand-* overrides via inline style on the root. Keep SSR deterministic — no Math.random in derivation.

4 · Deprecate

  • Inline hex values in buttonStyles.ts → <Button variant="primary" size="xl" />
  • Inline style objects in AddServiceForm → <Input />, <Select />
  • Hardcoded focus:border-purple-400 → shadow-focus token

5 · Never introduce

  • New fonts — stay on Geist.
  • Radii between 10 and 20 — pick one side.
  • Tinted shadows — warm-neutral rgba only.
  • Arbitrary Tailwind values like [13px] or [#abc] — use tokens.
