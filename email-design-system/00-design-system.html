Pattern Library

Reusable Section Patterns

CONFIRMATION SUMMARY

Info card with border-bottom header, label+value metadata rows, status badge. Used in: demo-confirmation, demo-reminder.

MESSAGE PREVIEW BLOCK

Tinted background card with quoted message content. Border-left accent. Used in: contact-received-user, contact-received-admin.

CALENDAR ADD ROW

Three ghost chip buttons side by side (Google, Outlook, Apple/ICS). Used in: demo-confirmation.

STAR RATING ROW

5 individual star link cells, each with direct rating param in URL. Table-based for Outlook compatibility. Used in: review-request.

SUPPORT FOOTER LINKS

Reschedule · Cancel · Support inline links separated by dots. Reused in all demo templates.

SECURITY NOTICE BLOCK

Muted tinted block with lock icon and 'if you didn't request this' safety note. Used in: password-reset.
